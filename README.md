# Loan Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a loan application will be approved or rejected using machine learning techniques.

Banks receive thousands of loan applications, and manually evaluating them can be slow and inefficient. This project demonstrates how machine learning can help automate loan approval decisions based on applicant data.

---

## 📊 Dataset Features
The dataset contains information such as:

- Gender
- Marital Status
- Dependents
- Education
- Self Employed
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area

---

## 🧠 Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Encoding Categorical Variables
5. Model Training
6. Model Evaluation

---

## 🛠 Technologies Used

- Scikit-learn

---

## 📈 Goal
Build a machine learning model that predicts loan approval based on applicant information.
